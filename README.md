#  AutobotWA – WhatsApp Automation Landing Page
A modern, responsive, and conversion-focused landing page built with **Next.js** and
**Tailwind CSS**, showcasing the power of **AutobotWA**, a tool designed to automate
repetitive WhatsApp workflows.
#  Project Purpose
The goal is to build a professional landing page that clearly explains AutobotWA’s automation
features while demonstrating strong UI/UX and frontend engineering skills.
# Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Vercel v0 Components
- Custom UI/UX Enhancements
# UI/UX Design Philosophy
- Minimal grey background theme
- Neon/turquoise CTA highlights
- Clean spacing & typography
- Smooth hover effects & micro interactions
- Conversion-focused layout
# Sections Included
# Navbar
Includes Features, Testimonials, Pricing, FAQ.
# Hero Section
Strong headline, subheadline, CTA buttons, dashboard image.
# Features Section
Auto-Replies, Scheduler, CRM Sync, Workflow Automation, Broadcast Tool, Templates.
# Testimonials
User reviews & trust elements.
# Pricing
Free, Pro, Business tiers with comparison design.
# FAQ
Answers to setup, automation, integrations, and usage questions.
# Responsiveness
Fully responsive across mobile, tablet, and desktop.
